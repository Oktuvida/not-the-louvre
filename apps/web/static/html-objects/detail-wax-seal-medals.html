Wax Seal Medals — Detailed Spec

CSS-rendered medals replacing emoji 🥇🥈🥉

Key design points:
  • Scalloped edge — irregular waxy polygon via clip-path, looks like a pressed wax seal
  • Inner circle — smooth border-radius circle with the rank number, slight transparent border
  • Metallic gradient — gold (#f4c430 → #d4a017), silver (#c0c0c0 → #a8a8a8), bronze (#cd7f32 → #a0622e)
  • Stamped highlight — subtle radial gradient overlay simulating light hitting the wax surface
  • No ribbons or cords — just the seal itself, clean and simple
  • Three sizes: large (1st place), standard (podium), small (inline/detail panel)
