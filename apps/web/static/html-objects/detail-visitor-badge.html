Visitor Badge — Detailed Spec

"HELLO MY NAME IS" badge replacing the plain "Signed in as" card

Key design points:
  • "HELLO MY NAME IS" — classic visitor name badge layout with colored header band
  • 5 color variants — red, blue, green, orange, purple — deterministically chosen from user ID hash
  • Avatar circle — 52px, dark border, in the white body area next to the nickname
  • Handwritten nickname — Caveat font, large, as if someone wrote their name on the badge
  • Red header/footer strips — or blue, green, orange, purple depending on the user's assigned color
  • Rounded corners — 12px border-radius, like a real sticker badge
  • Tape attachment — translucent tape strip on top attaching the badge to the wall
  • Slight rotation — -2deg, as if stuck on casually
  • Logout button moves out — becomes a separate sticker button below the Gallery/Mystery nav buttons

Implementation notes:
  • Replaces the current PersistentNav.svelte signed-in card (lines 22-38)
  • Create a VisitorBadge Svelte component with props: nickname: string, avatarUrl?: string, userId: string
  • 5 color variants: red (#c84f4f), blue (#4a7fb5), green (#5a9a5a), orange (#d4874d), purple (#8b6aae)
  • Color is deterministic — derived from user ID hash (hashString(userId) % 5), so the same user always gets the same badge color
  • CSS-only — rounded container with overflow hidden, colored divs for header/footer, flex body
  • Tape attachment — same translucent tape style used on post-its and polaroids
  • Fallback avatar — when avatarUrl is null, show a simple person icon SVG on the beige background
  • Logout button moves out of the badge — becomes a small sticker button below the Gallery and Mystery nav buttons in the bottom-left corner
  • Rotation: slight -2deg tilt, consistent with the "objects stuck on the wall" aesthetic
  • Position: absolute top-left corner of the homepage overlay, same as current card
