Polaroid Cards — Detailed Spec

Non-ranked artwork cards for the gallery grid

Key design points:
  • Avatar in the white caption area — small circle (32px) next to the title/artist text, not floating on the corner
  • Handwritten font — Caveat for title and artist name, Fredoka for stats
  • Each card has unique personality — different rotation, different attachment method (tape or pin), different pin color
  • Rank badge — small circle on top-left corner for positions #4+
  • White border — thicker at bottom to hold caption + avatar (classic Polaroid proportions)

Implementation notes:
  • Rotation: Deterministic from artwork ID hash — range [-3, 3] degrees, avoiding 0 (never perfectly straight)
  • Attachment method: Deterministic from artwork ID hash — alternates tape/pin, tape position varies (center/left/right/double)
  • Pin color: Deterministic from artwork ID hash — cycles through blue/red/green/yellow
  • Hover: Scale 1.06, rotation snaps to 0, z-index rises above neighbors
  • Caption area: Avatar (32px circle) on the left, text block (title, artist, stats) on the right
  • No torn/ragged edges — clean, straight polaroid borders
