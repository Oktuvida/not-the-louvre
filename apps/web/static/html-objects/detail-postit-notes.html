Post-it Notes — Detailed Spec

Room description cards and navigation tabs

Key design points:
  • Clean edges — no ragged tears, no weird ripped effects. Just clean rectangles with optional subtle corner curls
  • Color per room: Yellow = Hall of Fame, Pink = Hot Wall, Blue = Mystery, Green = Your Studio
  • Handwriting font — Caveat for the body text (room descriptions), Fredoka for the room name label
  • Different attachment per room — tape center, tape corner, pin — each room uses a different one
  • Subtle corner curl — optional, clean triangular fold showing the wall behind, not torn paper

Room description cards — one per room

Where legends are enshrined forever in gilded glory. Only the highest-voted artworks earn their spot on these hallowed walls.

The freshest, most buzzed-about pieces right now. Trending artworks fighting for the spotlight — today's heat, tomorrow's legends.

Peek behind the curtain at anonymous creations. Vote on raw talent — no names, no bias, just pure art appreciation.
Your Studio
Your personal corner of the museum. All your masterpieces (and happy little accidents) gathered in one place.

Implementation notes:
  • CSS-only — no canvas rendering needed; pure CSS backgrounds, pseudo-elements, and shadows
  • Room model drives styling: each room in rooms.ts gets a postItColor and attachment property
  • Corner curls are optional — some rooms have them, some don't, for variety
  • The curl reveals the wall — the triangular cutout matches the page background color (#b8956e or the gallery-bg.webp texture)
  • Tabs reuse the same color system — just smaller padding, no curl, used as navigation
