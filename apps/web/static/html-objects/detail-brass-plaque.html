Brass Plaque — Detailed Spec

Gallery title sign, screwed into the museum wall

Key design points:
  • Metallic gradient — diagonal sweep from bright gold to dark gold and back, simulating brushed brass
  • Four screws — one in each corner, each with a different slot rotation angle (never uniform)
  • Inner bevel — thin semi-transparent border inset 3px from the edge, simulating a beveled plate edge
  • Engraved text — dark brown with a subtle light text-shadow underneath (like light catching engraved letters)
  • Drop shadow — lifts the plaque off the wall slightly
  • Dark border — solid #8b6914 edge with rounded corners
