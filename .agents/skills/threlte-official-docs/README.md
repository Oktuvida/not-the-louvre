# Threlte Official Docs Skill

Repo-local skill for answering Threlte questions using only official Threlte
documentation.

## Purpose

This skill gives agents a strict, practical workflow for working with Threlte
without drifting into community lore or inferred behavior.

## Official Sources Only

- public docs on `threlte.xyz/docs`
- official repository `threlte/threlte`
- official docs source files under `apps/docs/src/content/learn` and
  `apps/docs/src/content/reference`

## Best Used For

- finding the correct official Threlte page
- separating Learn vs Reference material
- identifying which Threlte package owns a feature
- citing official documentation while implementing or reviewing code

## Companion Skills

- use with `svelte-code-writer` when editing or reviewing `.svelte` files
